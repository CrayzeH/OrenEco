Политика конфиденциальности

Настоящая политика описывает порядок обработки данных пользователей веб-приложения «OrenEco» и применяется ко всем разделам сайта.

Какие данные могут обрабатываться

Сайт может обрабатывать имя пользователя, адрес электронной почты, телефон, сведения учетной записи, данные формы обратной связи, результаты экологической экспертизы и техническую информацию, необходимую для работы веб-приложения.

Цели обработки данных

Данные используются для регистрации и авторизации пользователей, отображения личного кабинета, обработки обращений, сохранения результатов экспертизы, администрирования сайта и повышения удобства работы с экологической информацией.

Хранение и защита

Информация хранится в базе данных веб-приложения. Для снижения рисков используются разграничение доступа, авторизация пользователей и базовые меры защиты серверной части.

Передача данных

Персональные данные не передаются третьим лицам без необходимости. Исключение составляют случаи, связанные с выполнением требований законодательства или техническим обеспечением работы сервиса.

Обращения пользователей

Пользователь может направить вопрос через страницу «Контакты». Обращение сохраняется в административной панели и используется только для обработки запроса.
